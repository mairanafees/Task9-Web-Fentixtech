# 🪑 Furniture E-Commerce Website

A modern, responsive **Furniture E-Commerce Website** built using **HTML, CSS, and Vanilla JavaScript**, recreated from a Figma design with a focus on pixel-perfect UI and dynamic user interactions.

---

## Live Demo

🔗 task9-web-fentixtech.vercel.app

---


# 📌 Project Overview

This project recreates a furniture e-commerce landing page from Figma while implementing essential JavaScript features such as dynamic rendering, filtering, searching, and persistent cart functionality.

The application follows a clean folder structure and uses only HTML, CSS, and Vanilla JavaScript without any frameworks.

---

# ✨ Features

## 🎨 UI

* Pixel-perfect Figma implementation
* Modern furniture-themed design
* Fully responsive layout
* Clean typography
* Smooth hover animations
* Mobile-friendly navigation

---

## ⚙️ JavaScript Functionality

### Dynamic Product Rendering

* Products stored inside a JavaScript Array of Objects
* Product cards generated dynamically
* No hardcoded HTML product cards

### Category Filtering

* Instant filtering
* No page reload
* Active category highlighting

### Product Search

* Real-time search
* Search by product name
* Case-insensitive filtering

### Shopping Cart

* Add products to cart
* Live cart counter
* Data saved using LocalStorage
* Cart persists after page refresh
